Myeck - модуль для быстрого создания Content Entity с графическим интерфейсом.
(идея появилась после ознакомления с модулем eck, а также после создания собственной
 Entity по примеру "Content Entity Example" и создания Entity через консоль)
 Это тестовая версия. Bundle пока не поддерживаются.

Для создания - в разделе structure выбрать "Configure My ECK". Потом вкладку "Creating New Entity"
После создания, модуль скачивается и устанавливается как обычный модуль.
Созданый модуль зависим от Myeck, но в тоже время это позволило много кода контролеров
перенести в главную сущность. Модуль легко расширяется, после его создания можно добавить
свои роуты, контроллеры, формы и т.д. Entity полностью интегрируется у Views,
т.е. можно вывести каждое поле по отдельности.
В главную таблицу Entity можно добавить любое количество полей.
Список возможных типов полей, форматеров и виджетов формируется автоматически из всех,
которые зарегистрированы в системе. (При тестировании в основном использовались поля
 из блоков Текст и Число, также тип Пользователь) Чтобы поле было показано на странице
"Список созданных Entity данного типа" (контроллер ContactListBuilder в Example),
достаточно поставить птичку "show in list"

В отличии от модуля "Content Entity Example" (там есть баг), в созданную
Entity легко добавить новые поля через менеджер полей.

Сама Myeck также является полноценной Content Entity и может быть использована для каких-нибудь целей.
